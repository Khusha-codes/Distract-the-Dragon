# Dragon’s Distraction 🐉🍎

This is a Scratch game where the player help cat to reach the green flag while avoiding a fast dragon.

## 🎮 How to Play
- Help cat to reach green flag
- Use apples to distract dragon

## 🏆 Goal
help cat to reach the flag before the dragon catches it.

- dragon-game.sb3 (Scratch project file)
